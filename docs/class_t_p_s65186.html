void TPS65186::setThermistorThresholds ( byte  coldThreshold,
byte  hotThreshold 
)

Set threshold values for high and low-temperature interrupts. If the diference between baseline temperature and threshold is equal or greater than the threshold value, an interrupt issued. Default is 0 and 50.

Parameters
coldThresholdThe threshold for low temperature. A number from 0 to 15 represents temperatures from -7 to 8 degrees
hotThresholdThe threshold for low temperature. A number from 0 to 15 represents temperatures from 42 to 57 degrees
Returns
None
